# Formulário de Pesquisa de Satisfação – IFTO

Atividade de desenvolvimento de um formulário digital de pesquisa de satisfação institucional para o Instituto Federal do Tocantins (IFTO).

## Sobre

Formulário web criado em HTML/CSS puro, sem dependências externas. Coleta dados estruturados de discentes, docentes, técnicos administrativos e comunidade em geral sobre a experiência com a instituição.

## Como usar

Basta abrir o arquivo `index.html` no navegador.

## Tecnologias

- HTML5
- CSS3
